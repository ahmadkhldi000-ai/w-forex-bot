# Build & Deployment Info

## Vercel Auto-Deploy
This project is configured for automatic deployment on Vercel.

- **Platform:** Vercel (Next.js Static Export)
- **Production URL:** https://wforexbot.vercel.app
- **Source:** GitHub (`ahmadkhldi000-ai/w-forex-bot`)
- **Branch:** `main` → Production
- **Root Directory:** `web/`
- **Build Command:** `npm run build`
- **Output:** `out/` (static export)

## How Auto-Deploy Works
Any push to the `main` branch automatically triggers a new production deployment on Vercel.
